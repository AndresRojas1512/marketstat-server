# Приложение для анализа зарплат на российском рынке труда

## Название проекта: MarketStat

## Описание идеи проекта
MarketStat – это аналитическая система для анализа зарплат на российском рынке труда. Проект предоставляет инструменты для сравнения компенсаций по профессиям, регионам и отраслям на основе анонимизированных данных.

## Описание предметной области
Система работает с агрегированными данными о занятости, включая зарплатные показатели, профессиональные квалификации и географическое распределение, с соблюдением конфиденциальности.

## Анализ аналогичных решений

| Критерий          | HH.ru | Росстат | LinkedIn | MarketStat |
|-------------------|-------|---------|----------|------------|
| Гибкость анализа   | Средняя | Высокая | Поверхностная | Высокая |
| Частота обновлений | Ежемесячно | Ежеквартально | Раз в год | Ежемесячно |
| Защита данных     | Строгая | Строгая | Строгая | Строкая |

## Обоснование целесообразности
Проект актуален из-за потребности в оперативной аналитике зарплатных трендов в условиях трансформации рынка труда.

## Описание ролей

1. **Гость**
   - Просмотр усредненных данных
   - Применение шаблонных фильтров

2. **Зарегестрированный пользователь**
   - Доступ к детализированным отчетам
   - Экспорт данных
   - Фильтрация по множеству параметров

3. **Администратор**
   - Управление пользователями
   - Импорт и очистка данных
   - Конфигурация системы

## Пользовательские сценарии

### 1. Гость

1. **Открытие главной страницы**  
   - Пользователь заходит на сайт MarketStat без авторизации

2. **Выбор параметров анализа**
    - Применение шаблонных фильтров:
        - выбор региона;
        - выбор професиональной области
        - выбор опыт работы
3. **Просмотр шаблонных статистик**
    - Выбор готовых решений (статистик/отчетов).

3. **Просмотр результатов**
4. **Ограниченный доступ**  
   - При попытке экспорта данных система предлагает зарегистрироваться

---

### 2. Авторизованный пользователь

1. **Авторизация**
   - Ввод данных учетной записи

2. **Настройка отчета**
   - В разделе "Аналитика" возможны следующие параметры для настройки статистических данных:
     - професиональная область;
     - специализация;
     - город;
     - область;
     - опыт работы;
     - период;
     - режим работы;
     - название работадателя

3. **Анализ данных**  
   - Система строит:
     - виртуальные таблицы для просмотра данных на основе параметров, выбранных пользователем.

4. **Экспорт результатов**  
   - выбор формата (csv);
   - сохранение шаблона анализа

---

### 3. Администратор

1. **Доступ к панели администрирования**  
   - Авторизация с правами администратора.

2. **Загрузка файла данных**
   - Подготовка данных для их загрузки.

3. **Обновление данных**
4. **Публикация данных**  
   - Подтверждение загруженных / обновленных данных.
   - Запуск пересчета агрегированных показателей
5. **Добавление новых статистик**
   - загрузка новых статискик (отчетов) данных.
